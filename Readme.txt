1. Instalar Node js, express js, visual studio code, Xampp y Postman
2. Clonar el repositorio: https://github.com/pablogarciagatateresa/PruebaParqueadero.git
3. Iniciar Xampp con Mysql y Apache
4. Restaurar base de datos que se encuentra en la carpeta del proyecto dentro de PHPmyAdmin en el navegador web.
5. Ejecutar el comando node app.js en la aplicación
6. Abrir navegador y abrir localhost:3000, colocar rutas según archivo routes.js
7. Abrir Postman y ejecutar get y post según rutas del archivo routes.js
